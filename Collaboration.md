# Simple GitHub Collaboration Instructions
```
1. Clone the repo:
   git clone <repo-url>

2. Move into the folder:
   cd <repo-name>

3. Get the latest updates:
   git pull origin main

4. Create your own branch:
   git checkout -b <your-branch>

5. Make your changes to the files.

6. Save, add, and commit:
   git add .
   git commit -m "your update"

7. Push your branch:
   git push -u origin <your-branch>

8. Open a Pull Request on GitHub.

9. After your PR is merged, update again:
   git checkout main
   git pull origin main
```
